# A-la-recherche-du-temps-perdu 追忆似水年华

> “The screen prints 'Hello U' instead of 'Hello World' because U are my World”

一个充满诗意的交互式网页实验，将编程的理性与浪漫的隐喻编织在一起。通过代码动画、动态背景和可翻阅的语录，探索“Hello World”之外的情感表达。

## ✨ 特性

- **交互式开场**：点击手势触发“Hello U!”的逐字浮现
- **代码动画**：HTML 文档中的第 9 行逐字符“键入”出 `Hello U!`
- **场景过渡**：从赛博感渐变动画平滑过渡到温暖的米白色世界
- **语录集**：包含 4 条编程风格的浪漫短句，支持前后翻阅
- **响应式设计**：适配移动端与桌面端，代码显示随窗口尺寸调整
